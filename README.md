# OMR-V1 — Calibração Visual

Esta versão recomeça o projeto do ponto correto.

## Objetivo
Provar visualmente se a etapa de visão computacional funciona:

1. abrir câmera ou enviar imagem;
2. detectar os 4 marcadores pretos;
3. corrigir a perspectiva;
4. desenhar a grade sobre as 52 questões.

## Importante
Esta versão **não corrige prova** e **não exporta planilha**.  
Ela serve apenas para verificar se a grade cai exatamente sobre as bolhas.

Quando isso estiver perfeito, a leitura automática será simples e confiável.
