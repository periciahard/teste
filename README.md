# OMR V2 - Calibração Visual Integrada

Esta versão corrige a V2 anterior: agora os controles aparecem junto com a imagem corrigida e a grade muda em tempo real.

## Como usar
1. Envie uma foto do cartão ou abra a câmera.
2. Clique em processar.
3. Ajuste os controles até a grade azul ficar em cima das bolhas.
4. Clique em salvar calibração.
5. Envie um print para conferência.

Ainda não corrige prova; é apenas calibração visual.
